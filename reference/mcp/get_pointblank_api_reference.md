# get_pointblank_api_reference


Get API reference for Pointblank validation methods and common patterns.


``` json
{
  "tool": "get_pointblank_api_reference",
  "arguments": {
    "category": "validation_methods"
  }
}
```


# Parameters


`category: string = "validation_methods"`  
Category: 'validation_methods', 'thresholds', 'common_patterns', or 'all'
